Label Dialog

User's Guide > Dialog Reference > Label Dialog

Time
Label start time.
End
Label end time. This area is valid only for region labels.
Symbol
The symbol of the label. The contents are displayed in the waveform canvas.
Data
The data of the label. The contents are not displayed in the waveform canvas.
Channel
The channel of the label. Drawing position of the label changes based on this channel. This area is introduced in version 0.8.4. This channel information is lost in saving as a label file whose format doesn't support channel information.
