Title
Title of the song.
Artist
Artist name of the song.
Album
Album name.
Album Artist
Artist name of the album.
Genre
Genre list which is supported by MP3 format.
ID3v2
The version of ID3v2 tag which widely used in an MP3 file. This field is valid for file formats supporting ID3v2.
Composer
Composer name.
Lyricist
Lyricist name.
Producer
Producer name.
Engineer
Engineer information.
Release
Release information.
Tempo
Tempo information. The BPM value is mostly used.
Track
Track number.
Total Tracks
The number of total tracks.
Disc
Disc number.
Total Discs
The number of total discs.
Copyright
Copyright information.
Source
Source related information.
ISRC
International Standard Recording Code.
Software
Software information that the file is created by.
Subject
Subject.
Comment
Comment.
